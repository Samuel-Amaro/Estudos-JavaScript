Pesquisa Linear

Na ciência da computação, a pesquisa linear ou pesquisa sequencial é um método para localizar um valor-alvo em uma lista. Ele verifica sequencialmente cada elemento da lista para o valor de destino até que uma correspondência seja encontrada ou até que todos os elementos tenham sido pesquisados. A pesquisa linear é executada no pior tempo linear e faz no máximo n comparações, onde n é o comprimento da lista
